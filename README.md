# Data-Cleaning-Project-Marketing-Campaign-Dataset
This project focuses on data cleaning and preprocessing, to demonstrate essential data quality steps. The goal was to transform raw data into a cleaner, more reliable version ready for analysis.

# Objective
The main task was to apply standard data cleaning procedures using either Python or Excel, including:
Identifying and handling missing values
Removing duplicate rows
Standardizing text values
Converting date formats to a consistent type
Renaming column headers for clarity and uniformity
Checking and correcting data types


# Tool Used: 
Microsoft Excel

# Steps I Performed
Checked for Duplicates
Selected all columns and used Excel filters to check for duplicate rows.
Result: No duplicates found.

Renamed Columns
Renamed Cust. ID column header to Customer_ID for better readability and uniformity.
Standardized Birth Year
Cleaned and standardized the Customer Birth Year column to ensure consistent formatting and correct data type.

Handled Missing Values
Filled missing values in the Income column using the median value, maintaining statistical balance.
Standardized Enrollment Date
Used Excel’s ISNUMBER and date formatting functions to ensure all Customer Enrollment Dates were valid and in a consistent format.

Tools and Techniques Used
Microsoft Excel
Excel Functions: ISNUMBER(), filters, “Remove Duplicates”, “Find & Replace”

# Outcome

The final dataset is clean, consistent, and ready for further exploratory or predictive analysis.
All records are now free from duplicates and
missing values have been treated appropriatel
